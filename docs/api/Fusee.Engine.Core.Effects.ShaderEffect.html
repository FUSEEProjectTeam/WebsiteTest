Class ShaderEffect

A ShaderEffect contains a string for each, the vertex, fragment and geometry shader and a set of render states. Use this if you want to write the shader code on your own. The values of uniform variables you defined (ParamDecl) can be set using SetFxParam<T>(String, T).
