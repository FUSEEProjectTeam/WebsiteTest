Enum AxisBoundedType

Symbolic value describing if and how the values produced by this axis are bound.

Namespace: Fusee.Engine.Common
Assembly: Fusee.Engine.Common.dll
Syntax
public enum AxisBoundedType

Fields

Name Description
Constant

The axis values are bound by constant values. The values can be read from MaxValueOrAxis and MinValueOrAxis.
OtherAxis

This axis' values are bound by the values from other axes (and may possibly change over time). The bounding axes' Ids can be read from MaxValueOrAxis and MinValueOrAxis (cast to integer).
Unbound

The axis values are not bound.
