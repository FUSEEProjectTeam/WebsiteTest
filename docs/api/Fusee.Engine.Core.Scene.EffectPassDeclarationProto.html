Struct EffectPassDeclarationProto

A "proto" effect pass declaration contains the vertex and geometry shader source code as well as a partial Fragment Shader, that is completed in a pre-pass, depending whether we render forward or deferred. It also contains a RenderStateSet declaration for the rendering pass declared by this instance.
