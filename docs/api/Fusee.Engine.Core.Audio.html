Remarks

Supported file formats are: mp3, ogg, wav, flac, aiff, au, raw, paf, svx, nist, voc, ircam, w64, mat4, mat5 pvf, htk, sds, avr, sd2, caf, wve, mpc2k, rf64

Audio can be either buffered completely or be played as a stream.

E.g. : IAudioStreamImp myAudio1;

myAudio1 = Audio.Instance.LoadFile("AssetStorage/Music.ogg").

myAudio1.play();
