Use this class as a base class for implementing connectivity to whatever windows system you intend to support. Inherit from this class, make sure to call the constructor with the window handle to render on, implement the Run method and call the DoInit, DoUnload, DoRender and DoResize methods at appropriate incidences. Make sure that _width and _height are set to the new window size before calling DoResize.
